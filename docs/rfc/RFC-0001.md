## 3. Vision

### [RFC-0001:C-VISION] Core Vision (Normative)

PhaseOS is a governance system that ensures software development proceeds through well-defined phases with explicit gates.

The core principle: **No implementation work may proceed without a normative RFC in the appropriate phase.**

This is the fundamental invariant of PhaseOS governance.
